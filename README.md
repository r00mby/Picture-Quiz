# Picture-Quiz
An Android quiz application developed using Java in Android Studio. The app challenges users with image-based multiple-choice questions and provides instant feedback on their answers.

# Features

* Image-based quiz
* Multiple-choice questions
* Instant feedback for each answer
* Simple and user-friendly interface
* Lightweight Android application

# Technologies Used

* Java
* Android Studio
* XML
* Gradle

# How to Run

1. Clone or download this repository.
2. Open the project in Android Studio.
3. Let Gradle sync the project.
4. Build and run the application on an Android emulator or a physical device.

# Project Structure

* app/ – Application source code
* gradle/ – Gradle configuration files
* build.gradle.kts – Project build configuration
* settings.gradle.kts – Project settings
